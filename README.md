# SOLAPI PHP 예제 리스트

해당 예제는 Purplebook, Nurigo 등의 SOLAPI 마이사이트 등과 100% 호환되는 예제 리스트입니다.  
각각의 사용 환경에 맞게 예제 폴더를 참고하여 진행해보세요!

## 실제 개발연동 시 추가해야 할 의존성

### Composer

```bash
composer require solapi/sdk
```
